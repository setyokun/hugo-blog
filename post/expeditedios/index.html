Cara mengajukan Expedited iOS Apps agar segera Release di Appstore

Sep 02, 2019

Hey bro ..do! Pagi ini saya akan share gimana cara mengajukan Expedited iOS Apps agar segera release di Appstore

1 . Pertama, masuk ke page ini Expedite iOS , kemudian kalian akan diarahkan untuk login terlebih dahulu dengan akun itunesconnect yang kalian punya. Setelah masuk akan masuk ke page seperti dibawah ini :

2 . Pada Contact Information input Phone Country Code dan Phone Number, untuk Name, Email, dan Organization biasanya sudah default ke isi.

Pada App Information input App Name dan Apple ID of App, serta pilih Platformnya iOS.

notes : Apple ID of App bisa dilihat di Appstore aplikasi yang kamu punya

3 . Pada Description pilih Select a reason sesuai dengan kebutuhan, disini saya mencontohkan critical bug, setelah itu input explanation yang relevan dengan reason yang tadi kamu pilih, kemudian tinggal klik Send

dan Tinggal menunggu hasil expedite, jika hasil expedite diterima oleh tim apple, aplikasi kamu akan release lebih cepat (kurang dari 12 jam)dari biasanya (maximal 2 hari)

Sekian sharing singkat tentang expedite iOS dari gua, thanks bro ! do!
