Apa aja yang harus divalidasi / cek saat Testing API ?

Jul 15, 2019

Saat Testing API ada beberapa hal yang perlu di validasi dari response api yang kita request, sung cek gambar dibawah ini :

"status" : 200
    {
        "id": 5,
        "email": "setyokun@gmail.com",
        "first_name": "Setyo",
        "last_name": "Kun",
    }

1 . Cek Status dari Response API tersebut (ex : 200, 201, 404, dll)

2 . Cek Attribute / Property apa aja yang ada dan pastikan attribute sesuai dengan user story / requirement
- Harus ada attribute id, email, first_name, dan last_name

3 . Cek Value
- Harus ada value 2, setyokun@gmail.com, Setyo, Kun

4 . Cek Tipe Data dan Length Value
- Isi Value 2 - tipe data int - length 2
- Isi Value setyokun@gmail.com - tipe data string - length 30
- Isi Value Setyo- tipe data string - length 30
- Isi Value Kun - tipe data string - length 30
